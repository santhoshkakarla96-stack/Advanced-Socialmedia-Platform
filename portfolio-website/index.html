Security Layers

🔒
JWT + Refresh Tokens
Short-lived 7d access tokens with 30d refresh token rotation. Auto-refresh on 401 via Axios interceptor.
🚦
Rate Limiting
100 req/15min globally. Auth endpoints limited to 10 attempts with express-rate-limit and custom error messages.
🧱
Helmet Headers
CSP, HSTS, X-Frame-Options, X-Content-Type-Options, Referrer-Policy all configured for production.
🔑
Account Lockout
5 failed login attempts triggers a 2-hour lockout with progressive counter reset on success.
💉
Injection Protection
express-mongo-sanitize prevents NoSQL injection. xss-clean strips dangerous HTML. hpp blocks parameter pollution.
🐳
Non-root Docker
Production containers run as a dedicated nodejs user (uid 1001) — never as root.
