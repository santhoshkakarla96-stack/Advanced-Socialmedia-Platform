# Portfolio Website

A single-page portfolio showcasing the SocialSphere internship project.

## Live Preview
Open `index.html` in any browser — no build step required.

## Sections
- Hero with live terminal animation
- Tech stack grid (12 technologies)
- Core features breakdown (6 features)
- System architecture diagram
- Testing strategy with coverage bars
- CI/CD pipeline visualisation
- Security layers overview

## Deployment Options
- **GitHub Pages**: push to `gh-pages` branch, enable Pages in repo settings
- **Netlify**: drag `portfolio-website/` folder onto netlify.com/drop
- **Vercel**: `vercel --cwd portfolio-website`

## Customisation
1. Replace `yourusername` in GitHub links
2. Update `mailto:you@example.com` with your email
3. Update stats (tests count, coverage %) to match your actual results
